Simple MTS Merge - a simple program for merge .MTS video files. The program 
was written for my retired father. Communication is hard for him with a computer
and the program should be as simple as possible. Although the program was 
written using the Qt, it was written and tested only on Windows. Adapting it to 
GNU/Linux will not amount to labor.

Author: Andrei ivnitskii (ivnish.dev@gmail.com)

web: https://github.com/ivnish/simple-mts-merge
